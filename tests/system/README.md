# Системные тесты для проекта deb_scripts

Системные тесты предназначены для проверки работоспособности основных команд и функций в реальной системной среде, но изолированной через Docker-контейнер. Они отличаются от юнит- и интеграционных тестов тем, что проверяют реальное взаимодействие с системными ресурсами.

## Назначение

Системные тесты проверяют:

- Доступность системных команд (apt, curl, sysctl, iptables и др.)
- Возможность выполнения системных операций (чтение/запись файлов, управление сетью)
- Корректность установки зависимостей
- Работу в реальной среде Linux с учетом специфики контейнеров
- Взаимодействие с системными ресурсами и файловой системой
- Функционирование системных утилит и сервисов в изолированной среде

## Структура тестов

Тесты разбиты по следующим категориям:

- `test_system_commands.py` - базовые команды и системные операции
- `test_*_service_system.py` - тесты конкретных сервисов (BBR, Docker, Fail2Ban и др.)

## Запуск тестов

Для запуска всех системных тестов:

```bash
docker compose run --rm test pytest -m system -v
```

Для запуска конкретного теста:

```bash
docker compose run --rm test pytest tests/system/test_docker_service_system.py -v
```

Для запуска медленных тестов (с меткой `slow`):

```bash
docker compose run --rm test pytest -m "system and slow" -v
```

## Особенности тестирования в контейнере

- Не все системные команды доступны в контейнере (например, модули ядра)
- systemd может быть недоступен или иметь ограниченную функциональность
- Некоторые права доступа ограничены
- Тесты должны учитывать специфику работы в изолированной среде
- Взаимодействие с системными интерфейсами и службами может быть ограничено

## Требования к системным тестам

- Использовать метку `@pytest.mark.system`
- Обрабатывать ситуации отсутствия команд или ограниченного доступа
- Проверять работоспособность в реальных условиях, а не использовать моки
- Учитывать возможные различия между контейнерной и полноценной системной средой
- Проверять функциональность системных утилит и сервисов в реальной среде
